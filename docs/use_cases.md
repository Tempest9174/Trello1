# ユースケース / 操作フロー

実際の利用場面を想定した代表的な操作の流れです。

---

## UC-01 タスクを新規作成する

1. タスク一覧画面で「＋ タスクを追加」をクリックする
2. タイトルを入力する（必須）
3. 必要に応じて優先度・期限日・説明を設定する
4. 「保存」をクリックするとタスクが一覧に表示される

---

## UC-02 タスクの状態を更新する（進行中・完了）

1. タスク一覧画面でタスクをクリックして詳細画面を開く
2. 状態を「TODO → IN_PROGRESS → DONE」に変更する
3. 「保存」をクリックすると一覧に反映される

---

## UC-03 完了したタスクを削除する

1. 削除したいタスクをクリックして詳細画面を開く
2. 「タスクを削除」ボタンをクリックする
3. 確認ダイアログで削除を承認するとタスクが削除される（取り消し不可）
