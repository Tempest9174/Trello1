## 1. このアプリについて

本アプリは、タスクを一覧で管理できるシンプルなWebアプリケーションです。
タスクにタイトル・説明・優先度・期限日・状態を設定し、一覧で確認・更新できます。

---

## 2. できること・できないこと

### 2.1 必須機能

| 要件ID | 機能名 | 内容 |
|--------|--------|------|
| FR-01 | タスク一覧表示 | 登録されたタスクを一覧で表示する |
| FR-02 | タスク作成 | タイトル・説明・優先度・期限日を入力してタスクを作成する |
| FR-03 | タスク編集 | タスクの内容を更新する |
| FR-04 | タスク削除 | 不要なタスクを削除する |

**FR-01 タスク一覧表示**

登録されているタスクをposition順に一覧表示します。各タスクのタイトル・優先度・状態・期限日を一目で確認できます。

**FR-02 タスク作成**

タイトル（必須）・説明・優先度・期限日を入力して新しいタスクを作成します。優先度のデフォルトは「MEDIUM」、状態のデフォルトは「TODO」です。

**FR-03 タスク編集**

タスクをクリックして詳細画面を開き、タイトル・説明・優先度・期限日・状態を変更できます。

**FR-04 タスク削除**

不要なタスクを削除します。削除前に確認ダイアログを表示します。

---

### 2.2 このアプリではできないこと（対象外）

- ログイン・パスワード管理（認証機能なし）
- ファイルの添付
- 複数人での同時編集・リアルタイム同期
- スマートフォン・タブレットへの最適化

---

### 2.3 利用環境の制約

| 項目 | 内容 |
|------|------|
| 対応ブラウザ | Google Chrome（最新版）・Microsoft Edge（最新版） |
| 推奨画面サイズ | 横幅1280px以上のパソコン画面 |
| 利用環境 | ローカル環境での動作 |
| データの保持 | サーバーを停止してもデータは消えません |

---

### 2.4 非機能要件

| 要件ID | カテゴリ | 内容 |
|--------|---------|------|
| NFR-01 | パフォーマンス | 通常操作の応答は2秒以内 |
| NFR-02 | セキュリティ | SQLインジェクション・XSSなどの基本的なセキュリティ対策を実施する |
| NFR-03 | データ保全 | サーバーを停止してもデータベースのデータは消滅しない |
| NFR-04 | ブラウザ対応 | Chrome・Edge の最新版で正常に動作する |

## 3. バリデーション仕様

| 対象 | 項目 | ルール |
|------|------|--------|
| タスク | タイトル | 必須・最大200文字 |
| タスク | 説明 | 任意・最大1000文字 |
| タスク | 期限日 | 任意・日付形式（YYYY-MM-DD） |
| タスク | 優先度 | 必須・HIGH / MEDIUM / LOW のいずれか（デフォルト：MEDIUM） |
| タスク | 状態 | 必須・TODO / IN_PROGRESS / DONE のいずれか（デフォルト：TODO） |
| 削除操作 | 全般 | 削除前に確認ダイアログを表示する |

---

## 4. 受け入れ基準（完成の定義）

### FR-01 タスク一覧表示

- タスクがposition順に一覧表示される
- 各タスクのタイトル・優先度・状態・期限日が表示される

### FR-02 タスク作成

- タイトルを入力して作成ボタンを押すとタスクが一覧に追加される
- タイトルが空のときはバリデーションエラーが表示される
- ページを再読み込みしてもデータが残っている

### FR-03 タスク編集

- タスクをクリックすると詳細画面が開く
- 編集して保存するとデータが更新され一覧に反映される

### FR-04 タスク削除

- 削除ボタンを押すと確認ダイアログが表示される
- 削除後、タスクが一覧から消える
