## 1. このアプリについて

本アプリは、タスクをカード形式で管理できるWebアプリケーションです。
「やること」「進行中」「完了」のような列を作り、カードを並べたり移動させたりしながら作業を管理できます。
複数のボード（作業スペース）を切り替えて使うことができます。

---

## 2. できること・できないこと

### 2.1 必須機能

| 要件ID | 機能名 | 内容 |
|--------|--------|------|
| FR-01 | ボード管理 | 複数のボード（例：仕事用・個人用）を作成・切り替え・削除できる |
| FR-02 | リスト管理 | ボード内に列（リスト）を追加・名前変更・削除・並び替えができる |
| FR-03 | カード管理 | カードにタイトル・説明・期限日・優先度・ラベルを設定できる |
| FR-04 | ドラッグ操作 | カードをマウスで掴んで別の列や別の位置に移動できる |
| FR-05 | ラベル管理 | 色と名前をつけたラベルを作成し、カードに貼り付け・取り外しができる |

**FR-01 ボード管理**

ボードはアプリの「作業スペース」です。「仕事用」「個人用」のように用途ごとに複数作ることができ、ホーム画面からいつでも切り替えられます。ボードのタイトルは後から変更でき、不要になったボードは削除できます（削除するとボード内のすべてのデータも消えます）。

**FR-02 リスト管理**

リストはボード内の「列」です。「やること」「進行中」「完了」のように、作業の状態に合わせて自由に作れます。リストの名前はクリックして変更でき、不要なリストは削除できます。列の順番はドラッグ操作で入れ替えることができます。

**FR-03 カード管理**

カードは1件のタスクを表します。カードには以下の情報を設定できます。

- **タイトル**（必須）：タスクの名前
- **説明**（任意）：タスクの詳細メモ
- **期限日**（任意）：カレンダーから日付を選択
- **優先度**（高・中・低）：デフォルトは「中」
- **ラベル**（任意）：色付きタグを複数付与できる

**FR-04 ドラッグ操作**

カードをマウスで掴んで別の列へ移動したり、同じ列内で上下に並び替えることができます。列そのものも左右にドラッグして並び順を変えられます。移動後の状態はページを更新しても保持されます。

**FR-05 ラベル管理**

ラベルは色と名前を自由に設定できる「色タグ」です。ボードごとに作成・管理し、カードに貼り付けたり取り外したりできます。1枚のカードに複数のラベルをつけることができます。

### 2.3 このアプリではできないこと（対象外）

- ログイン・パスワード管理（1人で使う前提のため、認証機能はありません）
- ファイルの添付
- 複数人での同時編集・リアルタイム同期
- スマートフォン・タブレットへの最適化

---

## 3. ユースケース / 操作フロー

実際の利用場面を想定した代表的な操作の流れです。

### UC-01 ボードを新規作成して使い始める

1. ホーム画面で「＋ 新しいボード」をクリックする
2. ボード名を入力して作成する
3. ボード画面に遷移し、「＋ リスト追加」でリスト（列）を追加する
4. リスト内の「＋ カード追加」をクリックしてタスクを登録する

### UC-02 カードを作成してタスクを登録する

1. ボード画面のリスト内「＋ カード追加」をクリックする
2. タイトルを入力する（必須）
3. 必要に応じて優先度・期限日・ラベル・説明を設定する
4. 「保存」をクリックするとカードがリストに表示される

### UC-03 カードを別のリストへ移動する（ドラッグ＆ドロップ）

1. 移動したいカードをマウスでドラッグする
2. 移動先のリスト（列）にドロップする
3. 位置が自動保存され、ページを再読み込みしても反映されている

### UC-04 ラベルを作成してカードに付与する

1. カードをクリックしてカード詳細画面を開く
2. ラベル欄の「＋ ラベルを追加」をクリックする
3. 「＋ 新しいラベルを作成」で色と名前を設定して作成する
4. 作成したラベルにチェックを入れるとカードに付与される

### UC-05 完了したタスクのカードを削除する

1. 削除したいカードをクリックしてカード詳細画面を開く
2. 「カードを削除」ボタンをクリックする
3. 確認ダイアログで削除を承認するとカードが削除される（取り消し不可）

## 4. バリデーション仕様

入力データの制限・検証ルールです。

| 対象 | 項目 | ルール |
|------|------|--------|
| ボード | タイトル | 必須・最大100文字 |
| リスト | 名前 | 必須・最大100文字 |
| カード | タイトル | 必須・最大200文字 |
| カード | 説明 | 任意・最大1000文字 |
| カード | 期限日 | 任意・日付形式（YYYY-MM-DD）・過去日も入力可 |
| カード | 優先度 | 必須・「高」「中」「低」のいずれか（デフォルト：中） |
| ラベル | 名前 | 必須・最大50文字 |
| ラベル | 色 | 必須・プリセットカラーから選択 |
| 削除操作 | 全般 | 削除前に確認ダイアログを表示する・削除後は取り消し不可 |

## 5. 各機能の詳細説明

### カードの「優先度」について

カードには「高・中・低」の3段階で優先度を設定できます。優先度はカードの一覧にバッジとして表示されるため、重要なタスクをひと目で確認できます。デフォルト（初期値）は「中」です。

### カードの「期限日」について

カードに期限日を設定すると、カードに日付が表示されます。期限が近いタスクを視覚的に把握するのに役立ちます。期限日の設定は任意です。

### 削除操作について

ボード・リスト・カードは削除できます。削除した場合、その中にあるデータも一緒に消えます（例：ボードを削除すると、そのボード内のリストとカードもすべて消えます）。削除は取り消せませんのでご注意ください。

## 7. 受け入れ基準（完成の定義）

各機能が「完成した」とみなせる操作上の条件です。

### FR-01 ボード管理

- [+ 新しいボード] ボタンを押してタイトルを入力すると、一覧に新しいボードが表示される
- ボードのタイトルをクリックして書き換えると、画面に反映される
- ボードを削除すると、そのボード内のすべてのデータも一緒に消える
- ページを再読み込みしてもボードのデータが残っている

### FR-02 リスト管理

- リストを追加するとボード画面に新しい列が表示される
- リストのタイトルをクリックして書き換えられる
- リストを削除するとその列内のカードもすべて削除される
- リストを左右にドラッグして並び替えができる
- ページを再読み込みしてもリストのデータが残っている

### FR-03 カード管理

- [+ カード追加] からカードを作成すると列に表示される
- カードをクリックするとポップアップが開き、タイトル・説明・期限日・優先度・ラベルを編集できる
- [保存] ボタンを押すと変更が一覧にも反映される
- ページを再読み込みしてもカードの内容が残っている

### FR-04 ドラッグ操作

- カードを掴んで別の列にドロップすると、その列に移動する
- カードを同じ列内の別の位置にドラッグして順番を変えられる
- ページを再読み込みしても移動後の位置が保持されている
- 列を掴んで左右にドラッグして並び順を変えられる

### FR-05 ラベル管理

- ラベルを新規作成するとき、名前と色を設定できる
- カードのポップアップからラベルをカードに付与・取り外しできる
- 1枚のカードに複数のラベルをつけられる
- ラベルを削除するとそのラベルを付けていたカードからも外れる

## 8. システム構成（概要）

> データベースのテーブル設計・E-R図は [er_diagram.md](er_diagram.md) を参照してください。

本アプリは以下の3層で構成されています。

```
┌─────────────────────────────────────┐
│  ブラウザ（お客さまが使う画面）       │
│  タスクの表示・操作を担当する         │
└────────────────┬────────────────────┘
                 │ データのやり取り
┌────────────────▼────────────────────┐
│  サーバー（データ処理を行う部分）     │
│  保存・更新・削除の処理を担当する     │
└────────────────┬────────────────────┘
                 │ 保存・読み出し
┌────────────────▼────────────────────┐
│  データベース（データを保存する場所）  │
│  ボード・リスト・カード等を永続保存    │
└─────────────────────────────────────┘
```

### 保存されるデータの種類

ボードの中に複数のリストがあり、各リストの中に複数のカードがある構造です。ラベルはボード単位で作成し、カードに自由に付与できます。ボードを削除すると配下のリスト・カードもすべて自動で削除されます。
