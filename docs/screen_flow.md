# 画面遷移・設計書

---

## 1. 画面一覧

| 画面ID | 画面名 | 種類 | 説明 |
|--------|--------|------|------|
| S-01 | タスク一覧画面 | ページ | アプリ起動時に最初に表示される |
| S-02 | タスク詳細 | モーダル | タスクをクリックすると S-01 の上に重なる |

---

## 2. 画面遷移図

```
┌──────────────────────┐
│   タスク一覧画面（S-01）│  ← アプリ起動時に表示
└──────────┬───────────┘
           │ タスクをクリック
           ▼
┌──────────────────────┐
│  タスク詳細（S-02）   │  ← モーダル（S-01 の上に重なる）
└──────────────────────┘
```

---

## 3. タスク一覧画面（S-01）

```
┌────────────────────────────────────────────────────────────────────────┐
│  タスク管理アプリ                                                        │  ← ヘッダー
├────────────────────────────────────────────────────────────────────────┤
│  [ ＋ タスクを追加 ]                                                     │
├──────────────────────┬──────────────────────┬──────────────────────────┤
│  未着手 (2)          │  進行中 (1)          │  完了 (1)                │  ← カラムヘッダー
│                      │                      │                          │
│  ┌──────────────┐    │  ┌──────────────┐    │  ┌──────────────┐        │
│  │ [HIGH]       │    │  │ [MEDIUM]     │    │  │ [HIGH]       │        │  ← タスクカード
│  │ ログイン実装  │    │  │ DB設計レビュー│    │  │ 環境構築     │        │
│  │ 期限: 5/30   │    │  │ 期限: 5/20   │    │  │              │        │
│  └──────────────┘    │  └──────────────┘    │  └──────────────┘        │
│                      │                      │                          │
│  ┌──────────────┐    │                      │                          │
│  │ [LOW]        │    │                      │                          │
│  │ テスト作成    │    │  ← カードをここへ    │  ← ドラッグして移動      │
│  └──────────────┘    │     ドロップで移動    │                          │
│                      │                      │                          │
└──────────────────────┴──────────────────────┴──────────────────────────┘
```

**この画面でできること：**
- タスクをステータス別（未着手 / 進行中 / 完了）の3カラムで確認する
- タスクカードをドラッグ＆ドロップして別カラムへ移動（ステータス変更）
- タスクをクリックして詳細・編集モーダルを開く
- [＋ タスクを追加] ボタンで新しいタスクを作成する

---

## 4. タスク詳細モーダル（S-02）

```
┌───────────────────────────────────────────┐
│  [× 閉じる]                               │
│                                           │
│  タイトル                                  │
│  ┌──────────────────────────────────────┐  │
│  │ ログイン機能の実装                    │  │
│  └──────────────────────────────────────┘  │
│                                           │
│  優先度: [HIGH ▼]  （HIGH・MEDIUM・LOWから選択）  │
│                                           │
│  状態: [TODO ▼]  （TODO・IN_PROGRESS・DONEから選択）│
│                                           │
│  期限日: [ 2026-05-30 ]                   │
│                                           │
│  説明                                     │
│  ┌──────────────────────────────────────┐  │
│  │  （ここにメモを書く）                 │  │
│  └──────────────────────────────────────┘  │
│                                           │
│  [保存]                  [タスクを削除]   │
└───────────────────────────────────────────┘
```

**この画面でできること：**
- タイトル・説明・優先度・状態・期限日を編集して [保存] で確定する
- [タスクを削除] でタスクを削除する（確認ダイアログあり）
- [× 閉じる] または画面外をクリックして閉じる
