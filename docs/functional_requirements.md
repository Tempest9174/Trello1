# 機能要件（詳細）

---

## 1. 機能要件一覧

| 要件ID | 機能名 | 内容 |
|--------|--------|------|
| FR-01 | タスク一覧表示 | 登録されたタスクを一覧で表示する |
| FR-02 | タスク作成 | タイトル・説明・優先度・期限日を入力してタスクを作成する |
| FR-03 | タスク編集 | タスクの内容（タイトル・説明・優先度・期限日・状態）を更新する |
| FR-04 | タスク削除 | 不要なタスクを削除する |

---

## 2. 各機能の詳細説明

### FR-01 タスク一覧表示

登録されているタスクをposition順に一覧で表示します。各タスクにはタイトル・優先度バッジ・状態・期限日が表示されます。

### FR-02 タスク作成

「＋ タスクを追加」ボタンからタスクを作成します。タイトルは必須です。

設定できる項目：
- **タイトル**（必須）：タスクの名前（最大200文字）
- **説明**（任意）：タスクの詳細メモ
- **優先度**（必須）：HIGH / MEDIUM / LOW（デフォルト：MEDIUM）
- **期限日**（任意）：日付で指定
- **状態**（必須）：TODO / IN_PROGRESS / DONE（デフォルト：TODO）

### FR-03 タスク編集

タスクをクリックして詳細画面を開き、以下の情報を更新できます。
- タイトル・説明・優先度・期限日・状態

保存ボタンを押すとデータベースに反映され、一覧にも反映されます。

### FR-04 タスク削除

削除ボタンを押すと確認ダイアログを表示します。削除後は取り消しできません。

---

## 3. バリデーション仕様

| 対象 | 項目 | ルール |
|------|------|--------|
| タスク | タイトル | 必須・最大200文字 |
| タスク | 説明 | 任意・最大1000文字 |
| タスク | 期限日 | 任意・日付形式（YYYY-MM-DD） |
| タスク | 優先度 | 必須・HIGH / MEDIUM / LOW のいずれか |
| タスク | 状態 | 必須・TODO / IN_PROGRESS / DONE のいずれか |
| 削除操作 | 全般 | 削除前に確認ダイアログを表示する |

---

## 4. APIエンドポイント一覧

| メソッド | パス | 説明 |
|---------|------|------|
| GET | `/api/tasks` | タスク一覧取得（position順） |
| POST | `/api/tasks` | タスク作成 |
| PUT | `/api/tasks/{id}` | タスク更新 |
| DELETE | `/api/tasks/{id}` | タスク削除 |
